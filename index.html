React Roadmap PRO 2026

Um caminho completo, focado em prática e projetos reais, pensado para o Artioli: dashboards, e-commerces, SaaS e interfaces modernas.

0. Fundamentos de Web (HTML, CSS, JS)

Base sólida

Objetivo: Ser capaz de criar qualquer layout responsivo e interativo usando apenas HTML5, CSS3 e JavaScript puro.

HTML

  • Tags semânticas
  • Estrutura de documentos
  • Formulários e validação
  • Acessibilidade básica

CSS

  • Flexbox & Grid
  • Layouts responsivos
  • Transitions & animations
  • Variáveis CSS (custom properties)

JavaScript

  • Tipos, funções, escopo
  • DOM básico
  • Eventos
  • ES6+ (let/const, arrow functions)

Projeto sugerido: Landing page responsiva + mini app (To-Do, Pomodoro ou calculadora) usando apenas HTML, CSS e JS.

1. JavaScript Avançado

Nível React

Objetivo: Entender JavaScript em nível suficiente para escrever aplicações React limpas e performáticas.

Conceitos

  • Closures
  • Event loop e call stack
  • Imutabilidade
  • Higher-order functions

ES6+ avançado

  • Destructuring
  • Spread/Rest
  • Modules (import/export)
  • Async/await, Promises

Prática

  • Manipulação de arrays (map, filter, reduce)
  • LocalStorage / SessionStorage
  • Fetch API

Projeto sugerido: Mini dashboard em JS puro consumindo uma API pública.

2. React Core (Fundamentos)

React moderno

Objetivo: Dominar a base do React moderno com componentes funcionais e JSX.

Básico

  • JSX e renderização
  • Componentes funcionais
  • Props e composição
  • Listas e keys

Estado

  • useState
  • Renderização condicional
  • Eventos em componentes

Estilos

  • CSS Modules
  • CSS global
  • Estrutura base de pastas

Projeto sugerido: To-Do List premium + contador + mini dashboard de hábitos.

3. Hooks & State Management

Essência do React

Objetivo: Usar hooks para controlar fluxo de dados, interações e estados globais.

Hooks principais

  • useState, useEffect, useRef
  • useMemo, useCallback
  • useContext, useReducer

Conceitos

  • Controlled vs uncontrolled
  • Lifting state up
  • Compartilhamento de estado

Custom hooks

  • useLocalStorage
  • useTheme
  • useFetch

Projeto sugerido: Sistema de tema global (Dark/Light/Creative) + hooks reutilizáveis.

4. Routing & APIs

Apps reais

Objetivo: Criar aplicações multi-página e integradas com APIs reais.

React Router

  • Rotas básicas e aninhadas
  • Rotas dinâmicas
  • Rotas protegidas

APIs

  • Fetch / Axios
  • Loading e erro
  • Paginação e busca

UX

  • Skeletons
  • Empty states
  • Mensagens de feedback

Projeto sugerido: App de busca (GitHub, filmes ou clima) com múltiplas páginas e estados de loading.

5. Arquitetura Profissional em React

Escalabilidade

Objetivo: Estruturar projetos grandes usando padrões modernos de arquitetura.

Organização

  • Feature-based / domain-based
  • Atomic Design
  • Separação de responsabilidades

Camadas

  • components/, hooks/, services/
  • utils/, stores/
  • Aliases de import

Ferramentas

  • ESLint & Prettier
  • Padronização de código
  • Scripts de build

Projeto sugerido: Reescrever um dashboard (financeiro ou estudos) com arquitetura profissional.

6. Redux Toolkit & State Global

Apps grandes

Objetivo: Controlar estados complexos em aplicações médias e grandes.

Redux Toolkit

  • Slices
  • Store única
  • Thunks assíncronos

RTK Query

  • Queries
  • Mutations
  • Cache automático

Boas práticas

  • Selectors
  • Normalização de dados
  • Divisão por domínio

Projeto sugerido: App com autenticação + dashboard usando Redux Toolkit.

7. TypeScript para React

Padrão de mercado

Objetivo: Tipar componentes, hooks e serviços para aumentar segurança e clareza.

Básico

  • Tipos primitivos
  • Interfaces & types
  • Union & intersection

React + TS

  • Tipar props
  • Tipar hooks
  • Tipar contexto

Avançado

  • Tipos genéricos
  • Tipos utilitários
  • Validação com Zod/Yup

Projeto sugerido: Biblioteca de componentes tipada + refatorar um projeto existente.
